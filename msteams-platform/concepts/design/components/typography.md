# <a name="typography"></a>文字体裁

文字体裁はクリアで、意図的にする必要があります。 重要な情報を強調する必要があり、混乱を避けるために、複数のフォントとサイズを使用しないようにしてください。 ローカライズと読みやすくするために、文のケースを使用し、すべての cap の使用を避けることをお勧めします。
