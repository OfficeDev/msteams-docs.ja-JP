# <a name="designing-your-microsoft-teams-app-with-basic-fluent-ui-components"></a>基本的な Fluent UI コンポーネントを使用して Microsoft Teams アプリを設計する

Teams アプリは通常、次の基本的な Fluent UI コンポーネントで構築されています。 可能な限りフラットに設計されています。これらのコンポーネントは、さまざまなユースケース、テーマ、画面サイズで動作します。

このページの図は、Teams でのコンポーネントの既定 (淡色) と暗いテーマの外観を示しています。

## <a name="microsoft-teams-ui-kit"></a>Microsoft Teams UI Kit

<a href="https://fluentsite.z22.web.core.windows.net/" target="_blank">FLUENT ui</a>に基づいて、MICROSOFT Teams ui キットには teams アプリの作成専用に設計されたコンポーネントとパターンが含まれています。 UI キットでは、ここにリストされているコンポーネントを直接デザインに取得して挿入することができます。また、各コンポーネントの使用方法に関するその他の例を参照してください。

> [!div class="nextstepaction"]
> [Microsoft Teams UI Kit (Figma) を取得する](https://www.figma.com/community/file/916836509871353159)
